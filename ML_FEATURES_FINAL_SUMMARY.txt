USER WORKFLOW
================================================================================

STEP 1: Install Dependencies
-----------------------------
python install_ml_features.py

STEP 2: Enable Pattern Recognition (Start Here)
------------------------------------------------
1. Open dashboard ‚Üí Configuration tab
2. Scroll to "ML & AI Features"
3. Click to expand section
4. Check "Pattern Recognition"
5. Set Pattern Weight to 0.2-0.3
6. Set Technical Weight to 0.7-0.8
7. Click "Save Configuration"
8. Bot now uses pattern recognition!

STEP 3: Add Sentiment (Optional)
---------------------------------
1. Check "Sentiment Analysis"
2. Enter news data path (if available)
3. Set Sentiment Weight to 0.15-0.25
4. Click "Save Configuration"

STEP 4: Collect Training Data
------------------------------
1. Run bot for 1-2 weeks
2. Click "View ML Statistics" to check sample count
3. Wait until 100+ samples collected

STEP 5: Train ML Model
-----------------------
1. Click "Train ML Model"
2. Wait for training (shows progress)
3. Check accuracy in status message
4. Model saved automatically

STEP 6: Enable ML Predictions
------------------------------
1. Check "ML Predictions"
2. Set ML Weight to 0.2-0.3
3. Set Minimum Confidence to 0.6
4. Click "Save Configuration"
5. Full ML features now active!

STEP 7: Monitor & Adjust
-------------------------
1. Click "View ML Statistics" regularly
2. Check performance metrics
3. Adjust weights based on results
4. Export data for analysis
