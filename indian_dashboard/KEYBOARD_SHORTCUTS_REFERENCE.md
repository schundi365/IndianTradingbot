# Keyboard Shortcuts Quick Reference

## Navigation

| Shortcut | Action |
|----------|--------|
| `1` | Go to Broker tab |
| `2` | Go to Instruments tab |
| `3` | Go to Configuration tab |
| `4` | Go to Monitor tab |
| `5` | Go to Trades tab |

## Search and Refresh

| Shortcut | Action |
|----------|--------|
| `S` | Focus search input |
| `Ctrl` + `F` | Focus search input (alternative) |
| `R` | Refresh current tab |

## Configuration Actions

| Shortcut | Action |
|----------|--------|
| `Ctrl` + `S` | Save configuration |
| `Ctrl` + `L` | Load configuration |
| `Ctrl` + `E` | Export configuration |

## Help and Modals

| Shortcut | Action |
|----------|--------|
| `Shift` + `H` | Show keyboard shortcuts help |
| `Esc` | Close modals/dialogs |

## General Navigation

| Shortcut | Action |
|----------|--------|
| `Tab` | Move to next element |
| `Shift` + `Tab` | Move to previous element |
| `Enter` | Activate button/link |
| `Space` | Activate button/checkbox |
| `Arrow Keys` | Navigate within components |

## Tips

- Press `Shift` + `H` anytime to view this help
- Keyboard shortcuts work from anywhere except when typing in input fields
- `Ctrl` shortcuts work even when typing in input fields
- All interactive elements are accessible via keyboard

## Mobile Gestures

| Gesture | Action |
|---------|--------|
| Swipe Left | Next tab |
| Swipe Right | Previous tab |
| Tap and Hold | Show tooltip |

## Accessibility Features

- **Screen Reader Support**: All content is announced properly
- **Focus Indicators**: Clear visual indicators for keyboard navigation
- **Skip Link**: Press `Tab` on page load to skip to main content
- **High Contrast**: Automatically adapts to OS high contrast mode
- **Reduced Motion**: Respects OS reduced motion preferences
- **Dark Mode**: Automatically adapts to OS dark mode preference

## Browser Compatibility

Works in:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile browsers

**Need help?** Press `Shift` + `H` to view the interactive keyboard shortcuts guide.
