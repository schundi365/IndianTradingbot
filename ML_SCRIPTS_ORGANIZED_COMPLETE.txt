================================================================================
                        WHAT EACH SCRIPT DOES
================================================================================

0_extract_from_mt5.py
  - Connects to MT5
  - Downloads historical data
  - Calculates indicators
  - Simulates trades
  - Creates data/training_data.csv
  
  Requirements: MT5 running, symbols available

1_extract_training_data.py
  - Reads trading_bot.log
  - Extracts real trade signals
  - Extracts actual outcomes
  - Creates data/training_data.csv
  
  Requirements: Bot has been running, 50+ trades

2_prepare_training_data.py
  - Cleans extracted data
  - Creates additional features
  - Handles missing values
  - Creates data/training_data_prepared.csv

3_train_ml_model.py
  - Trains XGBoost model
  - Performs cross-validation
  - Shows accuracy metrics
  - Saves to models/ml_signal_model.pkl

4_evaluate_model.py
  - Tests model performance
  - Shows confusion matrix
  - Displays feature importance
  - Generates recommendations

5_deploy_model.py
  - Backs up existing model
  - Updates bot configuration
  - Records deployment
  - Provides restart instructions
