TROUBLESHOOTING
================================================================================

If you don't see ML messages:
- Check if ML is enabled: ml_enabled = true in config
- Check if model is trained: models/ml_signal_model.pkl exists
- Wait for signals to be generated (may take time)

If you don't see drawdown messages:
- Drawdown is only logged when there's actual drawdown
- Peak equity updates when equity increases
- Wait for some trading activity

If bot won't start:
- Check for error messages in console
- Verify MT5 is running and logged in
- Check bot_config.json is valid JSON
