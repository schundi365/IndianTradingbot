================================================================================
HOW TO SWITCH TO M30 DEFAULT
================================================================================

Option 1: Delete Config (Gets New Default)
1. Stop bot
2. Delete bot_config.json
3. Start bot
4. Bot creates new config with M30 default

Option 2: Edit Current Config
1. Stop bot
2. Open bot_config.json
3. Change "timeframe": 1 to "timeframe": 30
4. Save file
5. Start bot

Option 3: Use Dashboard
1. Open http://localhost:5000
2. Go to Configuration
3. Change Timeframe to M30
4. Save
5. Restart bot
