# Git Commands for Initial Deployment

## Step 1: Initialize Git Repository
git init

## Step 2: Add all files
git add .

## Step 3: Create initial commit
git commit -m "Initial commit: MT5 Trading Bot v1.0.0

Features:
- Adaptive risk management system
- Split orders with multiple take profit levels
- 6 different trailing stop strategies
- Smart position sizing
- Trade filtering with confidence scores
- Comprehensive documentation
- Example configurations
- Test scripts

Ready for demo testing."

## Step 4: Create main branch
git branch -M main

## Step 5: Add remote repository (replace with your GitHub URL)
git remote add origin https://github.com/yourusername/mt5-trading-bot.git

## Step 6: Push to GitHub
git push -u origin main

## Step 7: Create first release tag
git tag -a v1.0.0 -m "Release v1.0.0 - Initial Release"
git push origin v1.0.0

## Additional Commands

# Check status
git status

# View commit history
git log --oneline

# Create a new branch for development
git checkout -b develop

# View remote
git remote -v

# Pull latest changes
git pull origin main

# Create a new feature branch
git checkout -b feature/new-feature

# Merge feature to main
git checkout main
git merge feature/new-feature

# Delete branch
git branch -d feature/new-feature

## GitHub Repository Setup

After pushing, configure on GitHub:

1. Go to repository Settings
2. Add description: "Intelligent MT5 trading bot for Gold & Silver with adaptive risk management"
3. Add topics: python, trading, metatrader5, algorithmic-trading, forex, gold, silver, automated-trading
4. Enable Issues
5. Enable Discussions
6. Add repository image (optional)
7. Create first release from v1.0.0 tag
