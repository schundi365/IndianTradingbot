üéØ FEATURE DETAILS
================================================================

1. SYMBOL-SPECIFIC TP LEVELS
   --------------------------
   What it does:
   - Each symbol can have custom TP levels
   - Overrides default tp_levels for that symbol
   - Falls back to default if symbol not specified
   
   Example:
   - XAGUSD uses [1.5, 2.5, 4.0] (wider for silver volatility)
   - EURUSD uses [1.2, 2.0, 2.5] (tighter for forex)
   - GBPUSD uses default [1.5, 2.5, 4.0] (not specified)
   
   When to use:
   - Volatile symbols need wider TPs (XAGUSD, GBPJPY)
   - Stable symbols need tighter TPs (EURUSD, USDCHF)
   - Customize based on symbol behavior


2. TRAILING TAKE PROFIT
   ---------------------
   What it does:
   - When position is in profit, TP trails price
   - TP moves further away as price moves in your favor
   - Never moves closer (only extends)
   
   How it works:
   - trailing_tp_ratio: 0.5 means TP trails at 0.5√ó ATR distance
   - Lower ratio = more aggressive (trails closer)
   - Higher ratio = more conservative (trails further)
   
   Example:
   - Entry: $2,600, TP: $2,630 (initial)
   - Price moves to $2,650
   - TP trails to $2,680 (extended)
   - Price moves to $2,700
   - TP trails to $2,730 (extended again)
   
   When to use:
   - Strong trending markets
   - When you want to maximize winners
   - When you don't want to manually adjust TPs


3. SCALPING MODE
   -------------
   What it does:
   - Closes positions quickly when profitable
   - Two exit conditions:
     a) Profit target reached (e.g., 0.5%)
     b) Max hold time reached (e.g., 30 minutes)
   
   How it works:
   - scalping_profit_target: Close when profit reaches X%
   - scalping_max_hold_time: Close after X minutes if profitable
   - Only closes profitable positions
   
   Example:
   - Entry: $2,600
   - Profit target: 0.5% = $13 profit
   - Max hold time: 30 minutes
   
   Scenario 1: Profit target reached in 10 minutes
   ‚Üí Position closes at $13 profit
   
   Scenario 2: 30 minutes passed, $8 profit
   ‚Üí Position closes at $8 profit (time limit)
   
   Scenario 3: 30 minutes passed, -$5 loss
   ‚Üí Position stays open (not profitable)
   
   When to use:
   - Fast-moving markets
   - When you want quick profits
   - When you want to reduce exposure time
   - M1 or M5 timeframes

SYMBOL-SPECIFIC TP LEVELS:
--------------------------
"symbol_tp_levels": {
    "SYMBOL": [TP1, TP2, TP3]
}

- SYMBOL: Must match MT5 symbol exactly (e.g., "XAUUSD")
- TP1, TP2, TP3: Must be in ascending order
- Values are multipliers of risk distance
- If symbol not specified, uses default tp_levels

Examples:
  "XAGUSD": [1.5, 2.5, 4.0]   // Wide TPs for silver
  "EURUSD": [1.2, 2.0, 2.5]   // Tight TPs for forex
  "GBPJPY": [2.0, 3.0, 5.0]   // Very wide for volatile pair


TRAILING TP:
------------
"enable_trailing_tp": true/false
"trailing_tp_ratio": 0.1 to 1.0

- enable_trailing_tp: Turn feature on/off
- trailing_tp_ratio: Distance TP trails from price
  - 0.1 = Very aggressive (trails very close)
  - 0.5 = Balanced (recommended)
  - 1.0 = Conservative (trails far away)

Examples:
  Aggressive: 0.3
  Balanced: 0.5
  Conservative: 0.7


SCALPING MODE:
--------------
"use_scalping_mode": true/false
"scalping_profit_target": 0.1 to 5.0 (%)
"scalping_max_hold_time": 5 to 120 (minutes)

- use_scalping_mode: Turn feature on/off
- scalping_profit_target: % profit to close position
- scalping_max_hold_time: Minutes to hold before closing if profitable

Examples:
  Quick scalping: 0.3%, 15 minutes
  Balanced: 0.5%, 30 minutes
  Patient: 1.0%, 60 minutes
