WHY M30 AS DEFAULT
================================================================================

Benefits of M30 Conservative Strategy:

✅ Higher Quality Signals:
   - Longer timeframe filters out noise
   - Multiple filters ensure quality
   - Better win rate expected

✅ Lower Risk:
   - Fewer trades = less exposure
   - Conservative position limits
   - Multiple safety filters

✅ Manageable:
   - 1-5 trades per day
   - Easy to monitor
   - Less stressful

✅ Stable:
   - Won't overload MT5 connection
   - Sustainable long-term
   - Lower API call frequency

✅ Profitable:
   - Focus on quality over quantity
   - Better risk/reward ratios
   - Compound gains over time

================================================================================
WHEN DEFAULT CONFIG IS USED
================================================================================

The default M30 configuration will be used when:

1. First Time Launch:
   - No bot_config.json exists
   - Bot creates new config with M30 defaults

2. Config File Deleted:
   - User deletes bot_config.json
   - Bot recreates with M30 defaults

3. Config File Corrupted:
   - bot_config.json is invalid JSON
   - Bot falls back to M30 defaults

4. Missing Config Keys:
   - Config file missing some settings
   - Bot merges with M30 defaults for missing keys
