   GEM TRADING BOT - WINDOWS EDITION

Thank you for downloading GEM Trading Bot!

This version includes ML (Machine Learning) signal prediction for enhanced
trading decisions. The ML model analyzes patterns and provides additional
confirmation for trade signals.

================================================================================
   QUICK START - 3 STEPS
================================================================================

STEP 1 - Install MetaTrader 5
   - Download MT5 from your broker's website
   - Install and login to your account
   - Enable Algo Trading in Tools > Options > Expert Advisors
   - Check "Allow algorithmic trading"
   - Keep MT5 running!

STEP 2 - Run the Bot
   - Double-click GEM_Trading_Bot.exe
   - Wait 30 seconds for dashboard to open
   - Browser opens automatically to http://localhost:5000

STEP 3 - Configure and Start
   - Click "Configuration" tab
   - Select symbols (XAUUSD, GBPUSD recommended)
   - Choose timeframe (M5 recommended)
   - Enable "Auto" for all settings
   - Click "Save Configuration"
   - Click "Start Bot"
   - Monitor your trades!

   ML FEATURES
================================================================================

   This bot includes Machine Learning capabilities
   - ML model analyzes 8 technical features
   - Provides BUY/SELL/NEUTRAL predictions
   - Confidence-based signal filtering
   - Works alongside technical analysis
   - Can be enabled/disabled in dashboard

   ML Settings in Dashboard
   - Enable ML - Turn ML predictions on/off
   - ML Confidence - Minimum confidence threshold (60% recommended)
   - Require Agreement - How many signals must agree

================================================================================
   TROUBLESHOOTING
================================================================================

   Dashboard won't open
   - Wait 30 seconds on first run
   - Try manually - http://localhost:5000
   - Check Windows Firewall (allow access)

   Bot won't start
   - Make sure MT5 is running
   - Verify MT5 is logged in
   - Check algo trading is enabled

   No trades executing
   - Lower confidence level to 40%
   - Check symbols are selected
   - Verify account has sufficient balance

   For more help, see TROUBLESHOOTING.md
