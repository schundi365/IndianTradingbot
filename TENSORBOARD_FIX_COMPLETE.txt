   WHAT WAS THE PROBLEM?
================================================================================

When you ran build_windows.bat, PyInstaller was failing with:

    ModuleNotFoundError: No module named 'tensorboard'

trading bot doesn't actually use it. PyInstaller was trying to include it,
causing the build to fail.

================================================================================
   TROUBLESHOOTING
================================================================================

IF BUILD STILL FAILS:
---------------------

1. Update PyInstaller:
   pip install --upgrade pyinstaller

2. Clear pip cache:
   pip cache purge

3. Reinstall dependencies:
   pip uninstall -y pandas numpy
   pip install pandas numpy

4. Check Python version (need 3.8+):
   python --version

5. Try building in console mode:
   - Edit build_windows.bat
   - Change --windowed to --console
   - Rebuild and check error messages

IF EXECUTABLE CRASHES:
----------------------

1. Check MT5 is installed and running
2. Check algo trading is enabled in MT5
3. Check Windows Firewall allows the exe
4. Check antivirus isn't blocking it
5. Try running as Administrator

IF FEATURES DON'T WORK:
-----------------------

1. Check trading_bot.log for errors
2. Use dashboard System Logs tab
3. Click "Check MT5 Connection"
4. Verify bot_config.json exists
5. See TROUBLESHOOTING.md
